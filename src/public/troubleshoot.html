MACHINE TROUBLESHOOTING

Screen is black / machine seems off

  • Press the power button once and wait ~60 seconds — network boot takes longer than a normal PC.
  • Check the monitor input (HDMI/DisplayPort) and that the monitor is on.

Stuck on a boot screen or an error about the network

  • Hold the power button 5 seconds to turn it off, wait 10 seconds, power on again.
  • Check the network cable is clicked in at both the wall and the machine.

Windows is slow or broken

  • These machines reset to a clean image on reboot — save anything you need somewhere else, then just restart the machine.

Still stuck?

  • Ask a staff member — mention the machine name shown above (also printed on the sticker).
